JCoffeeScript

JCoffeeScript is a java library that compiles CoffeeScript.

Jars

jCoffeeScript 1.6.1

Usage

from the command prompt:

echo "a = 1" | java -jar jcoffeescript.jar

      (function() {
          var a;
          a = 1;
       })();

Command Line (unix/windows):

java -jar jcoffeescript.jar < foo.coffee > foo.js

command line options:··

--bare - compile the javascript without top-level function safety wrapper.
